## 核心项目：费哲基准分析监管决策平台

**FacilityONE AI Baseline Analysis & Decision Platform**

### 项目背景

费哲（FacilityONE）是中国设施管理（FM）软件领域的领军企业。本项目由我与公司核心决策层共同主导，从0到1设计并交付一套面向资产业主与FM服务商的AI决策分析平台，定位为"中国设施管理领域的 Palantir"。

### 我的工作

- **产品架构设计**：定义完整的 L1/L2/L3 三级信息架构，覆盖5大模块、45个决策场景，历经多轮迭代至 v4 版本
- **原型交付**：独立完成45个场景页面的可交互 Dashboard 原型，每页包含详细数据、原因分析、优化建议三层结构，搭配 AI 助手面板
- **数据分析**：基于恒隆地产12个在营项目、267张真实 FM 数据库表，进行跨项目基准对比，识别关键运营异常节点
- **报告输出**：以 Palantir 风格设计数据诊断报告，将原始数据转化为可供管理层直接决策的结构化洞察
